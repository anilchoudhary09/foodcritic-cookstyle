🔒 5. barc_cops.rb - Static Engine (1,446 lines)

🚫 NEVER MODIFY THIS FILE

The barc_cops.rb file is the enforcement engine. It reads data from rules.rb and enforces all 36 BARC policies using RuboCop/Cookstyle.

Example: service_name Attribute Detection

The following patterns are now detected:

# Resource name detection (original) service 'sshd' do action :start end # service_name attribute detection (NEW) service 'my_ssh_service' do service_name 'sshd' # <-- Now detected! action :start end

Example: tags (Plural) Detection

# Single tag (original) tag 'barc' # Multiple tags (NEW) tags %w[barc unix role] # <-- Now detected! tags ['tag1', 'tag2'] # <-- Also detected!

Example: .NET PowerShell Pattern Detection (BARC001/002)

# PowerShell .NET DirectoryEntry user creation (NEW) powershell_script 'create_user' do code <<-EOH $ou = [ADSI]"WinNT://localhost" $user = $ou.Create("user", "myuser") # <-- BARC001 detected! $group = $ou.Create("group", "mygrp") # <-- BARC002 detected! EOH end

Example: Variable Tracking (BARC027/031)

# Variable holding MW package (NEW - tracked) mw_package = 'jboss-eap-7.0' # <-- Variable tracked package mw_package do # <-- BARC027 violation! action :install end # 1-liner resource (NEW - detected) package 'tomcat-8.5.0' # <-- BARC027 detected without block!

Example: BARC024 Regex Validation (NEW)

# metadata.rb validation patterns: MAINTAINER_PATTERN = /\A[^0-9`!@#\$%\^&*+_=]+\z/ # No special chars/numbers EMAIL_PATTERN = /\A[\w+\-.]+@[a-z\d\-]+(\.[a-z\d\-]+)*\.[a-z]+\z/i SOURCE_URL_PATTERN = /^(http(s)?):(\/\/)/i # Must be http/https # Also validates: source_url must contain cookbook name source_url 'https://github.com/org/my-cookbook' # ✓ Valid source_url 'https://github.com/org/other-repo' # ✗ Missing cookbook name

Example: BARC034 JSON Role Support (NEW)

# Now checks BOTH .rb and .json role files: # roles/my_role.rb default_attributes( 'enable_custom_sudoers_policies' => true # <-- BARC034 violation! ) # roles/my_role.json (NEW - now detected) { "default_attributes": { "enable_custom_sudoers_policies": true // <-- BARC034 violation! } }
