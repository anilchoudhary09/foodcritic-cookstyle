🔄 Cookstyle Workflow with Barclays Custom Cops

Purpose: This document provides visual flow diagrams showing the complete process from developer machine to CI/CD pipeline for using Cookstyle with Barclays custom BARC cops.

👨‍💻 Developer → 💻 Local Test → 📦 Git Push → 🔧 Jenkins → 📊 Reports

What Happens When You Run cookstyle .

cookstyle . │ ├──▶ Reads .rubocop.yml │ │ │ └──▶ require: ./barc_cops.rb │ │ │ └──▶ BarcRulesData.load_rules! reads rules.rb │ │ │ ├── @restricted_services (500+ exceptions) │ ├── @etc_whitelist (200+ exceptions) │ ├── @etc_blacklist (protected paths) │ └── @system_services (protected services) │ └──▶ Applies BARC001-BARC019 cops to your cookbook │ └──▶ Reports violations (with exception handling)

🔒 Exception Checking Logic

┌──────────────────────┐ │ Cookbook Resource │ │ e.g., service 'ntp' │ └──────────┬───────────┘ │ ▼ ┌──────────────────────┐ │ BARC017 Cop Fires │ │ "System service │ │ management" │ └──────────┬───────────┘ │ ▼ ┌────────────────────────────────────┐ │ Is 'ntp' in @system_services? │ └────────────────┬───────────────────┘ │ ┌──────────────┴──────────────┐ │ │ ▼ ▼ ┌─────────┐ ┌─────────┐ │ NO │ │ YES │ └────┬────┘ └────┬────┘ │ │ ▼ ▼ ┌────────────┐ ┌────────────────────────────┐ │ ✅ PASS │ │ Check @restricted_services│ │ No issue │ │ for exceptions │ └────────────┘ └─────────────┬──────────────┘ │ ┌─────────────┴─────────────┐ │ │ ▼ ▼ ┌──────────────────────┐ ┌──────────────────────┐ │ Cookbook in allowed │ │ Cookbook NOT in │ │ list OR list empty │ │ allowed list │ └──────────┬───────────┘ └──────────┬───────────┘ │ │ ▼ ▼ ┌────────────┐ ┌────────────┐ │ ✅ PASS │ │ ❌ FAIL │ │ Exception │ │ Violation │ │ Applied │ │ Reported │ └────────────┘ └────────────┘
Example:
@restricted_services = { 'ntp' => ['is_platform_ntp_cookbook', 'is_base_config'] }

my-app-cookbook trying to manage 'ntp' → ❌ VIOLATION (not in allowed list)
is_platform_ntp_cookbook trying to manage 'ntp' → ✅ PASS (in allowed list)

👮 BARC Cops Summary

Rule Description Exception Source
BARC001 Do not create local users @platform_cookbook_whitelist
BARC002 Do not create local groups @platform_cookbook_whitelist
BARC003 Do not modify root .ssh files No exceptions
BARC005 Do not modify protected /etc paths @etc_whitelist, @etc_blacklist
BARC006 Do not reboot/halt/shutdown No exceptions
BARC007 Do not modify SELinux No exceptions
BARC008 Do not kill processes No exceptions
BARC009 Do not modify firewall No exceptions
BARC011 No dangerous file removal No exceptions
BARC016 Use Chef resources (not shell) Convention only
BARC017 Do not manage system services @system_services, @restricted_services
BARC019 No find/sudo/chmod 777 No exceptions

⚡ Quick Reference Commands

Environment Command Purpose
Local cookstyle . Run all checks
Local cookstyle . --only Barclays Run only BARC cops
Local cookstyle . --autocorrect Auto-fix issues
Jenkins cookstyle . --format json --out report.json JSON output for pipeline
Debug cookstyle . --debug Verbose output

Main Flow Diagram

flowchart TD
    subgraph DEV["👨‍💻 Developer Machine"]
        A[Clone b-cookstyle-rules repo] --> B[Create/Edit Cookbook]
        B --> C{Run cookstyle locally}
        C -->|Violations Found| D[Fix violations]
        D --> C
        C -->|Clean| E[git push to GitHub]
    end

    subgraph GITHUB["📦 GitHub Repository"]
        E --> F[Webhook triggers Jenkins]
    end

    subgraph JENKINS["🔧 Jenkins Pipeline"]
        F --> G[Checkout Code]
        G --> H[Setup Environment]
        H --> I[Load barc_cops.rb]
        I --> J[barc_cops.rb reads rules.rb]
        J --> K[Run cookstyle with BARC cops]
        K --> L{Violations?}
        L -->|Yes| M[Generate JSON Report]
        L -->|No| N[✅ Build Success]
        M --> O[Parse & Categorize Violations]
        O --> P[Generate HTML Report]
        P --> Q{FAIL_ON_VIOLATIONS?}
        Q -->|true| R[❌ Build Failed]
        Q -->|false| S[⚠️ Build Unstable]
    end

    subgraph REPORTS["📊 Reports & Artifacts"]
        N --> T[Archive Artifacts]
        R --> T
        S --> T
        T --> U[Publish HTML Report]
        U --> V[Email Notification]
    end
